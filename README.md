
### Программа создает новый массив строк, содержащий только те элементы массива, длина которых < или = 3 символам.
• Процесс начинается с вызова метода Print, который запрашивает у пользователя ввод массива.
Затем введенная строка разбивается на массив строк с разделителя ", "

• Метод Filter сначала вычисляет количество строк, перебирая исходный массив строк.
Затем создается новый массив нужного размера, и в него копируются подходящие строки из массива.

• Затем новый массив выводится в консоль.
